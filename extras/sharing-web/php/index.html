1. Variables Scope

Scope pada PHP adalah jangkauan tertentu dimana variabel tersebut masih bisa digunakan/direferensikan, pada PHP sendiri terdapat 3 jenis scope.

  1. Local
  2. Global
  3. Static

Local Scope

Sebuah variabel yang dideklarasikan di dalam sebuah function hanya bisa diakses pada function tersebut. 

              
                function cetakAngka() { 
                  $x = 5; // variabel dengan scope local
                  echo "Nilai variabel x pada fungsi adalah : $x";
                  echo "
";
                }
                cetakAngka();
                // Menggunakan variabel x di luar fungsi akan menghasilkan error
                echo "Variable diluar fungsi adalah: $x";

Global Scope

Sebuah variabel yang dideklarasikan di luar function mempunyai jangkauan global dan hanya bisa diakses di luar function. 

              
                $x = 5; // variabel dengan scope global
                
                function cetakAngka() { 
                  // Menggunakan variabel x di dalam fungsi ini akan menghasilkan error
                  echo "Nilai variabel x pada fungsi adalah : $x";
                  echo "
";
                }
                cetakAngka();
                echo "Variable diluar fungsi adalah: $x";

Menggunakan keyword global

Sederhananya keyword global menjadikan variable di dalam function yang memiliki jangkauan lokal menjadi global. 

              
                $x = 5; // variabel dengan scope global
                
                function cetakAngka() { 
                  global $x; // membuat variabel x menjadi global dan dapat diakses pada fungsi ini

                  echo "Nilai variabel x pada fungsi adalah : $x";
                  echo "
";
                }
                cetakAngka();
                echo "Variable diluar fungsi adalah: $x";

Static

Adalah jenis variabel yang mempertahankan nilainya pada setiap pemanggilan fungsi. Untuk variabel normal, nilai dari variabel tersebut akan secara otomatis dihapus pada saat fungsi selesai dijalankan, dan akan dibuat ulang pada saat fungsi dipanggil. 

              
                function cetakAngkaIncreament() {
                  static $x = 0; // variabel ini nilainya dipertahankan
                  echo $x;
                  $x++; // disini dilakukan perubahan terhadap variabel x, yakni penambahan 1 angka
                }
                
                cetakAngkaIncreament();
                cetakAngkaIncreament();
                cetakAngkaIncreament();

2. Anonymous Function

Adalah fungsi yang tidak memiliki nama spesifik (anonim). Contoh berikut adalah pendeklarasian variabel yang isinya adalah anonymous function. 

              
                $cetakAngka = function(){
                  $x = 5; // local scope variable
                  echo $x;
                };

                $cetakAngka(); // 5

Anonymous Function dengan argumen/parameter. 

              
                $cetakAngka = function($angka){
                  echo $angka;
                };

                $cetakAngka(5); // 5
                $cetakAngka(); // error: too few arguments, karena fungsi ini membutuhkan argumen

Callback Function

Adalah sebuah function yang dijadikan sebagian argumen pada function lainnya. 

              
                // fungsi ini dideklarasikan untuk mengembalikan kalimat perintah (callback)
                function perintah($kalimat){
                    return $kalimat . "! ";
                }
                // fungsi ini dideklarasikan untuk mengembalikan kalimat pertanyaan (callback)
                function tanya($kalimat){
                    return $kalimat . "? ";
                }
                // menerima callback
                function cetakKalimat($format, $kalimat){
                    // parameter pertama kita jadikan sebagai function (menerima fungsi callback)
                    echo $format($kalimat);
                    echo '
';
                }

Cara pemanggilan callback function 

              
                cetakKalimat("perintah", "Sholat dulu yuk"); // Sholat dulu yuk!
                // mencetak kalimat perintah dengan konsep callback

                cetakKalimat("tanya", "Apakah Anda sudah sholat"); // Apakah Anda sudah sholat?
                // mencetak kalimat tanya dengan konsep callback
                
                cetakKalimat("callbackTidakAda", "Callback tidak ada"); // error; call to undefined function
                // percobaan 3 : mencetak kalimat yang callbacknya tidak didefinisikan

4. Die dan var_dump()

Die (sama dengan exit()) adalah builtin function dari PHP yang digunakan untuk menghentikan (terminate) ekseksui kode berikutnya (biasanya digunakan pada saat pengembangan). 

              
                echo 'Pesan ini dijalankan.';
                echo '
';
                echo 'Pesan ini masih dijalankan.';
                die(); // menghentikan eksekusi terhadap kode berikutnya
                echo 'Pesan ini tidak dijalankan.'; // kode ini tidak akan terkesekusi

Die dapat menerima parameter

Die dapat menerima sebuah argument/parameter berupa pesan, pesan apa? yaitu pesan yang disampaikan kepada pengguna ketika die dieksekusi. 

              
                fopen("diare.txt", "r") or die("File tidak ditemukan."); // menampilkan error
                echo '
';
                echo 'Good job, filenya ada.'; // file ini dieksekusi jika file ditemukan pada path

var_dump

var_dump adalah fungsi yang digunakan untuk mengetahui informasi detail sebuah variabel, var_dump biasa digunakan dalam pengembangan dan biasa disandingkan dengan die. 

              
                $angka = 5;
                $array_angka = [1, 2, 3];
                var_dump($angka); // int(5)
                echo '
';
                var_dump($array_angka); // array(3) { [0]=> int(1) [1]=> int(2) [2]=> int(3) }

var_dump dapat menerima lebih dari satu variabel 

              
                $angka = 5;
                $array_angka = [1, 2, 3];
                $huruf = 'ABC';
                var_dump($angka, $array_angka, $huruf); // int(5) array(3) { [0]=> int(1) [1]=> int(2) [2]=> int(3) } string(3) "ABC"
